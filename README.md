# Smart Home Automation System

Smart Home Automation System is a console-based C++ project developed using Object-Oriented Programming and File Handling concepts.
The system simulates a smart home environment where users can manage rooms, control appliances, operate lighting and security systems, 
control the water tank motor, and generate usage reports through a menu-driven interface.

## Features
- Password protected login system
- Room and appliance management
- Lighting control system
- Home security system
- Water tank motor control
- Usage reports for ON/OFF devices
- Permanent data storage using text files

## OOP Concepts Used
- Classes and Objects
- Inheritance
- Encapsulation
- Constructors and Destructors
- Getter and Setter Functions
- Aggregation
- Dynamic Object Handling
- File Handling
